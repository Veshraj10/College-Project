# FIFA World Cup Ultimate Quiz 2026

## Live Website

https://veshraj10.github.io/College-Project/

## Project Description

The FIFA World Cup Ultimate Quiz 2026 is a responsive web application developed using HTML, CSS, and JavaScript. The application tests users' knowledge of FIFA World Cup history through a set of multiple-choice questions. It includes a timer, score calculation, difficulty progression, a leaderboard, and a light/dark mode for a better user experience.

## Features

- Responsive design for desktop, tablet, and mobile devices
- 20 FIFA World Cup quiz questions
- 10-second timer for each question
- Automatic difficulty progression
- Score calculation
- Leaderboard using Local Storage
- Light and Dark mode
- Simple and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
